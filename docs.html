Fit.Person(name, gender, dob, weight, height, race)

People are at the heart of health care, which why the Person object is the heart of the library. The Person object stores the properties from which the Cardiovascular and Muscle objects obtain their properties for calculations, making it the most important object in the library.

Properties

  • name
  • gender
  • dob
  • height
  • weight
  • cardio
  • muscle

Usage

						
 // create a Person instance
 var me = new Person("John Doe", "male", new Date(1990, 3,17), 54, 154);
	
 // pick an equation from Person
 me.Predicted_TEE()['active'];  // 2537.0212935729905 (kcal)
	
 // or from one of its components
 me.cardio.HeartRateMax();  // 190.538 (bpm)

Person.getAge()

Returns the age of the Person in years. This method is used primarily to ensure the age used in calculations is accurate at the time of execution.

Usage

						
 // create a Person instance
 var me = new Person("John Doe", "male", new Date(1990, 3,17), 54, 154);
	
 me.getAge();  // 24.96

Person.netCaloricCost(mets)

Returns the caloric cost in kilocalories. METs can be obtained by manually entering the number or through the METs module

Note: Mets must be in MET form, not mL/kg/minute

Usage

							
 // create a Person instance
 var me = new Person("John Doe", "male", new Date(1990, 3,17), 54, 154);
	
 // pick an equation from Person
 me.netCaloricCost(300);  // 283.5 (kcal)

Person.bmiToBodyFat()

Returns an object of percentage of body fat for adults, and a child.

Usage

							
 // create a Person instance
 var me = new Person("John Doe", "male", new Date(1990, 3,17), 54, 154);
	
 // pick an equation from Person
 me.bmiToBodyFat();  
 {
 child: -0.196009441271428,
 adult: -0.219158504271645
 }

Person.skinfoldDb(sum)

Where sum is the result of a skinfold test, returns an object of percentage of body fat for many populations.

Populations

  • black
  • white
  • collegiateathlete:
    • black
    • white
  • child

Usage

							
 // create a Person instance
 var me = new Person("John Doe", "male", new Date(1990, 3,17), 54, 154);
	
 // pick an equation from Person
 me.skinfoldDb();  
 {
 black: 1.094301868944607,
 child: 19.375,
 collegiateathlete: {
 	black: -5.514,
 	white: -3.516 },
 white: 1.0833126542577594
 }

Person.dbAtTLCNS(db)

Where db is body density (g/mL), returns body density from hydrostatic weighing while total lung capacity without head submersion (TLCNS).

Usage

							
 // create a Person instance
 var me = new Person("John Doe", "male", new Date(1990, 3,17), 54, 154);
	
 // pick an equation from Person
 me.dbAtTLCNS(0.95);  // 0.959655

Person.bodySurfaceArea()

Where db is body density (g/mL), returns body surface in meters2.

Equations

  • Boyd: 0.03330 * Math.pow(weight,(0.7285-0.0188*Math.log(weight)))*Math.pow(height,0.3)
  • Costeff: (4*weight+7)/(90+weight)
  • DuBois: 0.0087184 * Math.pow(weight,0.425) * Math.pow(height,0.725)
  • Fujimoto: 0.008883 * Math.pow(weight, 0.444) * Math.pow(height, 0.663)
  • GehanGeorge: 0.0235 * Math.pow(weight, 0.51456) * Math.pow(height, 0.42246)
  • Haycock: 0.024265 * Math.pow(weight, 0.5378) * Math.pow(height, 0.3964)
  • Mosteller: Math.sqrt(weight*height)/60
  • Takahira: 0.007241 * Math.pow(weight, 0.425) * Math.pow(height,0.725)

Usage

							
 // create a Person instance
 var me = new Person("John Doe", "male", new Date(1990, 3,17), 54, 154);
	
 // pick an equation from Person
 me.bodySurfaceArea();
 {
 Boyd: 2.045653492421787,
 Costeff: 1.5486111111111112,
 DuBois: 1.8308664274017985,
 Fujimoto: 1.4725455220642767,
 GehanGeorge: 1.5368390922502115,
 Haycock: 1.5268447264673155,
 Mosteller: 1.5198684153570663,
 Takahira: 1.5206120160598762
 }

Person.bodyVolume(uww, rv, gv=100)

Where uww is underwater weight(kg), rv is residual volume (mL), gv is gastrointestinal volume (mL), and wd is water density in (g/mL) returns body volume in m3.

Usage

							
 // create a Person instance
 var me = new Person("John Doe", "male", new Date(1990, 3,17), 54, 154);
	
 // pick an equation from Person
 me.bodyVolume(50, 1200, 100);  // 0.05 (m3)
