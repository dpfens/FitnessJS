FitnessJS a collection of exercise physiology calculations

FitnessJS structures exercise physiology equations to allow you to focus more on the results of the calculations, rather than performing the calculations themselves. There are many exercise physiology equations in existence, but they were conducted using different units of measure, and by researchers of different nationalities. FitnessJS is a compilation of these researcher's efforts (sources included) for web developers to utilize.

Quick Start

The library is built upon 3 objects: Person, Cardiovascular, and Strength. Cardiovascular and Strength objects are the building blocks of the Person object.


// create a Person for me
me = new Person("John Doe", "male", new Date(1990, 3,17), 54, 154);
	
// pick an equation from Person
me.Predicted_TEE()['active'];  // 2537.0212935729905 (kcal)
	
// or from one of its components
me.cardio.HeartRateMax();  // 190.538 (bpm)
